 install flake8 with pip install flake8
 install pylint by downloading tarball from https://pypi.python.org/pypi?%3Aaction=search&term=pylint&submit=search and
  gunzip pylint-1.4.4.tar.gz
  tar xopf pylint-1.4.4.tar
  cd pylint-1.4.4
  sudo python setup.py install
  (installed in ~/packages)
  use flycheck-verify-setup to see what checkers are properly hooked
